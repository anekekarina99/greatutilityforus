# Keyword Plan — Pet Name Generator (Dog & Cat) (Bing-focused)

> Dokumen ini KHUSUS riset keyword Bing untuk cluster `dog cat name maker generator`.
> Sumber data: Bing autosuggest (alphabet-soup scrape) seed `dog cat name maker generator`.
> Status: draft riset, BELUM divalidasi volume. Validasi wajib di Microsoft Advertising Keyword Planner sebelum produksi.

## 1. Insight utama

Seed `dog cat name maker generator` itu **frasa janggal hasil scrape**, bukan cara orang asli mencari. Keyword nyata yang lebih sehat muncul di dalam dump:

- `cat name generator`, `dog name generator`
- `pet name generator`, `pet cat name generator`, `pet dog name generator`
- `name generator for cats`, `name generator for dogs`, `name generator for pets`

Ini **cocok sekali dibuat client-side**: kumpulan nama berkualitas + filter (jenis hewan, gender, gaya) + tombol "generate/acak" + favorit. Tanpa backend, tanpa upload. Selaras filosofi situs.

## 3. Sinyal sub-niche menarik

- **Warrior Cats** (`wc cat name generator`, `med cat name generator`): komunitas besar & loyal, intent sangat spesifik.
- **Minecraft** (`minecraft cat/dog name generator`): audiens gaming/anak.
- **D&D / OC** (`dnd pet name generator`, `cat oc name generator`): roleplay/kreatif.

Sub-niche ini lebih mudah di-rank daripada head term umum karena kompetisi lebih sempit.

## 4. Noise yang dibuang

- Frasa seed janggal: `dog cat name maker generator <huruf> online/free` (artefak alphabet-soup).
- Preposisi/phrase junk: `beneath`, `amid`, `circa`, `unto`, `upon`, `besides`, `alongside`, `despite`, `underneath`, `upside down`, `worth the cost`, `near me`.
- Programmer: `in python`, `in excel`, `github`.

## 5. Peringatan kompetisi

1. Head term `cat/dog name generator` dipenuhi situs besar (the spruce pets, rover, dll.).
2. Diferensiasi: **kualitas kurasi nama + filter bagus + UX cepat + sub-niche** (warrior cats, minecraft, dnd).
3. Tool tipe ini mudah dibuat → banyak pesaing; menang lewat kurasi & long-tail, bukan head term.
