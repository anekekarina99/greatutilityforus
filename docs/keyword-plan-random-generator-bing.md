# Keyword Plan — Random Generator (Bing-focused)

> Dokumen ini KHUSUS riset keyword Bing untuk cluster `random generator`.
> Sumber data: Bing autosuggest (alphabet-soup scrape) seed `random generator`.
> Status: draft riset, BELUM divalidasi volume. Validasi wajib di Microsoft Advertising Keyword Planner sebelum produksi.

## 1. Insight utama

`random generator` bukan satu tool tunggal. Ini lebih cocok menjadi **kategori / hub** yang berisi beberapa micro-tool:

- Random number generator
- Random wheel / spinner
- Random name picker
- Random password generator
- Random team / group generator
- Random list picker
- Random letter / word generator
- Dice roller
- Yes or no generator

Kalau semua dipaksa ke satu halaman, intent-nya terlalu melebar. Strategi yang lebih kuat: **1 intent = 1 URL**.

## 2. Cluster tool nyata

| Calon tool | Keyword pendukung di autosuggest | Catatan |

| Random number generator | `random generator numbers`, `random generator 1-10`, `random generator 1-100`, `random generator 0-100`, `random generator between two numbers`, `random generator of numbers` | Core dan universal, tapi kompetisi tinggi karena Google punya widget bawaan |
| Random wheel / spinner | `random generator wheel`, `random generator wheel spinner`, `random generator spin wheel`, `random generator wheel names`, `random generator number wheel` | Populer, tapi butuh UI animasi |

## 3. Sinyal bahasa / pasar

| Bahasa / pasar | Sinyal keyword | Implikasi |
|---|---|---|
| Ceko | `jmen a prijmeni`, `hesla`, `slov`, `ulic`, `jmen do hry` | Ada sinyal kuat untuk random names/password/words/address versi Ceko |

| Global / English | `number`, `wheel`, `names`, `password`, `teams`, `list` | Bisa jadi basis versi `en` |

Catatan: signal Ceko cukup menarik karena muncul berulang, tapi perlu validasi volume di Microsoft Advertising Keyword Planner sebelum diprioritaskan.

## 4. Noise yang dibuang

Jangan dipakai sebagai target:

- Preposisi / phrase junk: `beneath`, `amid`, `circa`, `unto`, `upon`, `beside`, `alongside`, `despite`, `underneath`, `outside the box`
- Brand / entitas acak: `post malone`, `xqc`, `9gag`, `49ers`, `6abc`, `9news`, `xfinity`, `53 bank`
- Frasa tidak relevan: `above and beyond lyrics`, `under armour`, `down syndrome`, `near me`, `till dawn`

Ini artefak autosuggest/alphabet-soup, bukan intent tool yang sehat.

## 5. Peringatan kompetisi

Cluster random generator lebih sulit daripada finance calculator:

1. Google sering punya widget bawaan untuk `random number generator`.
2. Pemain besar sudah kuat: `random.org`, `wheelofnames.com`, Google widget, dan berbagai calculator sites.
3. Banyak head keyword punya intent instan: user ingin langsung pakai tool, bukan membaca konten.

Strategi terbaik: jangan mulai dari head term `random generator` atau `random number generator`. Mulai dari long-tail spesifik dan bahasa non-Inggris.

## 6. Prioritas halaman yang disarankan

1. **Random Team Generator**  
   Target: `random generator for teams`, `random generator for groups`, `random generator kelompok`  
   Alasan: long-tail spesifik, cocok untuk classroom/workshop, lebih mudah dibedakan.

2. **Random Name Picker**  
   Target: `random generator names`, `random generator name picker`, `random generator from a list of names`  
   Alasan: kuat untuk raffle/giveaway/classroom.

3. **Random Password Generator**  
   Target: `random generator password`, `random generator for passwords`, `8 digit password generator`  
   Alasan: evergreen, tapi perlu pesan keamanan yang jelas.

4. **Random Wheel Spinner**  
   Target: `random generator wheel`, `random generator wheel spinner`, `spin wheel`  
   Alasan: demand besar, tapi effort UI lebih tinggi.

5. **Random Number Generator**  
   Target: `random generator numbers`, `random generator 1-100`, `random generator between two numbers`  
   Alasan: core, tapi sangat kompetitif dan dilawan widget Google.

## 7. Rekomendasi strategi

- Jadikan `random generator` sebagai **hub kategori**, bukan satu halaman tool utama.
- Buat tool dengan long-tail spesifik lebih dulu: `team/group`, `name picker`, `password`.
- Hindari head term terlalu awal karena kompetisinya tinggi.
- Untuk multi-bahasa, cek peluang Ceko dan Indonesia karena ada sinyal unik dari autosuggest.
- Tetap prioritaskan `compound interest calculator` lebih dulu jika tujuan utama monetisasi iklan, karena finance biasanya punya CPC lebih tinggi.
