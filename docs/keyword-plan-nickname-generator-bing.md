# Keyword Plan — Nickname Generator (Bing-focused)

> Dokumen ini KHUSUS riset keyword Bing untuk cluster `nickname generator`.
> Sumber data: Bing autosuggest (alphabet-soup scrape) seed `nickname generator`.
> Status: draft riset, BELUM divalidasi volume. Validasi wajib di Microsoft Advertising Keyword Planner sebelum produksi.

## 1. Insight utama

`nickname generator` punya **dua intent berbeda** yang harus dipisah:

1. **Gaming nickname / stylish text**: nama keren + font fancy + simbol untuk game (Free Fire, BGMI, PUBG, Fortnite, Roblox, CODM). Ini intent terbesar & paling spesifik.
2. **Nickname dari nama asli**: julukan/panggilan berdasarkan nama (`nickname generator based on name`, `from name`).

Keduanya **client-side friendly**: transformasi teks (unicode fancy fonts, simbol) + daftar pola julukan. Tanpa backend. Sangat selaras filosofi situs.

## 4. Noise yang dibuang

- Preposisi/phrase junk: `beneath`, `amid`, `circa`, `unto`, `upon`, `besides`, `alongside`, `despite`, `underneath`, `upside down`, `till the end`, `near me`.
- Brand/entitas acak: `home depot`, `qvc`, `xfinity`, `under armour`, `9gag`, `xqc`, `vrbo`, `vpn`, `0% apr`.
- Off-topic: `does nickname generator affect seo`, `down syndrome`.

## 5. Peringatan kompetisi

1. Niche gaming nickname & fancy text dipenuhi pemain kuat (nickfinder, lingojam, fancy text tools).
2. Diferensiasi: **UX cepat + banyak gaya font + copy 1-klik + preset per game + multi-bahasa**.
3. Head term `nickname generator` kompetitif; menang lebih mudah lewat **long-tail per game** (`free fire nickname`, `bgmi stylish name`).

## 6. Prioritas halaman yang disarankan

1. **Stylish Nickname / Fancy Text Generator** (BUILD-able client-side)
   Target: `nickname generator stylish`, `with symbols`, `with fancy text`, `copy and paste`
   Fitur: input teks → banyak varian font unicode + simbol + copy.

2. **Game Nickname Generator (preset per game)**
   Target: `nickname generator for games`, `free fire`, `bgmi`, `pubg`, `fortnite`, `roblox`
   Fitur: preset gaya per game, share/copy.

3. **Nickname from Name**
   Target: `nickname generator based on name`, `from name`
   Fitur: nama asli → daftar julukan.
